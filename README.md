This repository centralizes practical work for the **AI Principles and Technologies** course at ECI. It serves as a comprehensive portfolio demonstrating the implementation of key AI concepts and techniques through hands-on laboratories and a final project.

The repository explores fundamental AI topics:

-   **Neural Networks**: Building and training networks from scratch, implementing forward/backward propagation algorithms, and using Keras framework
-   **Decision Trees**: Developing entropy-based classification models using ID3 algorithm with scikit-learn
-   **Adversarial Search**: Implementing game theory strategies with minimax and alpha-beta pruning
-   **Knowledge-Based Systems**: Creating rule-based expert systems with inference engines
-   **Optimization Algorithms**: Applying heuristic search methods for problem-solving

Each laboratory combines theoretical foundations with practical implementation, using modern frameworks like Keras, scikit-learn, and pandas.

## ðŸ“š Laboratory Assignments

### Laboratory 1: Neural Networks

**Objective:** Model and solve problems using neural networks, implementing core algorithms from scratch.

**Topics:**

-   Neural network architecture fundamentals
-   Feed-forward algorithm implementation
-   Backpropagation learning algorithm
-   Keras framework for deep learning

**Key Deliverables:** Custom neural network implementation with gradient descent optimization, activation functions, and practical applications using Keras.

</br>

### Laboratory 2: Decision Trees

**Objective:** Develop classification models using decision trees and master scikit-learn framework.

**Topics:**

-   Entropy and information gain concepts
-   ID3 algorithm (Iterative Dichotomiser 3)
-   Tree construction and pruning techniques
-   Data preprocessing with pandas

**Key Deliverables:** Decision tree classifier built from scratch, scikit-learn implementation, and comprehensive data analysis using DataFrames.

</br>

### Laboratory 3: Adversarial Search

**Objective:** Implement game AI using adversarial search algorithms and game theory principles.

**Topics:**

-   Game theory fundamentals
-   Minimax algorithm for optimal play
-   Alpha-beta pruning optimization
-   Heuristic evaluation functions

**Key Deliverables:** Two-player game AI with minimax implementation, alpha-beta pruning for efficiency, and strategic decision-making systems.

</br>

### Laboratory 4: Knowledge-Based Systems

**Objective:** Design and implement rule-based expert systems with symbolic reasoning capabilities.

**Topics:**

-   Rule-based system architecture
-   Forward chaining (data-driven reasoning)
-   Backward chaining (goal-driven reasoning)
-   Inference engines and explanation facilities

**Key Deliverables:** Expert system with knowledge base, inference engine supporting both forward and backward reasoning, and explanation capabilities for reasoning chains.

## ðŸš€ Final Project: RepoAura

**RepoAura** is a hybrid AI system combining **rule-based reasoning** and **deep learning** to predict defect-prone software modules using NASA open-source datasets (CM1, JM1, KC1, KC2, PC1).

### System Architecture

-   **Rule-Based Filter**: Threshold-based classification (PASS/WARN/FAIL) using expert system
-   **Neural Network**: MLP for pattern detection in code quality metrics
-   **Optimization Engine**: Hill climbing for cost-effective refactoring recommendations

### AI Techniques Applied

1. **Neural Networks**: Defect prediction from historical software metrics
2. **Decision Trees**: Feature importance and module classification
3. **Knowledge-Based Systems**: Rule-based quality gates for code standards
4. **Search Algorithms**: Optimal refactoring paths with cost-benefit analysis
